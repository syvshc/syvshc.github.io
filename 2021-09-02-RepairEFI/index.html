因为半夜迷糊, 运行了


来删除我某文件夹下的全部文件. 删到一半才反应过来我干了些什么, 看着崩坏的图形界面, 我只想回到 Windows, 把 Arch Linux 的分区格式化, 再重新安装, 但是我发现, 我已经把 Windows 的 EFI 引导破坏了……

感谢 #archlinux-cn-offtopic 群的 Xiao Jerry!

  2. 点击右下角: 修复计算机
  3. 进入命令行, 运行 找到后用 A 添加
  4. 添加后退出命令行, 点击启动修复
  5. 完成
